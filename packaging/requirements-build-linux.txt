# Hash-pinned Linux build toolchain for the runtime bundles (Linux release
# lane). Versions are LOCKSTEP with requirements-build.txt (the macOS lane) —
# bump both together. Differences from the macOS file, by design:
#   * macholib is OMITTED — it is a macOS-only PyInstaller dependency.
#   * pyinstaller carries TWO hashes (manylinux2014 x86_64 + aarch64): the
#     bootloader is compiled, so each arch ships its own wheel. pip with
#     --require-hashes matches whichever wheel the runner downloads, so this
#     one file serves both linux/amd64 and linux/arm64 (glibc/manylinux
#     runners; musllinux/Alpine is intentionally not targeted).
# The pure-python deps (altgraph, packaging, pyinstaller-hooks-contrib,
# setuptools) are py3-none-any — identical hashes to the macOS file.
#
# Regenerate (on a Linux host):
#   pip download --only-binary :all: --no-deps -d wheels \
#     pyinstaller==<ver> altgraph==<ver> packaging==<ver> \
#     pyinstaller-hooks-contrib==<ver> setuptools==<ver>
#   pip hash wheels/*.whl
# Install:
#   pip install --require-hashes --no-deps -r requirements-build-linux.txt
